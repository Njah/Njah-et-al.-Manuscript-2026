Bismark commands

##Trimming reads

trim_galore --paired phred33 --fastqc --retain_unpaired --illumina <file names in pair>

###I-Preparing reference genome for bisulfite alignment using the following command:

bismark_genome_preparation --verbose  < path_to_genome_folder >

###II-Aligning BSeq reads to reference genome from (I) using the following command:

bismark --genome /path to/reference genome and trimmed reads/ -1 <read1> -2 <read2>

###Deduplication
deduplicate_bismark --paired --bam sample.bismark_bt2_pe.bam
 
## Methylation extractor

bismark_methylation_extractor --gzip --bedGraph --buffer_size 10G --cytosine_report --
genome_folder /path_to_file_folder/ sample_bismark_bt2_pe.deduplicated.bam
